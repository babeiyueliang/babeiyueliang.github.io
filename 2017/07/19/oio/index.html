##Node.js

Node.js是一个基于Chrome V8引擎的JavaScript运行时。Node.js使用高效、轻量级的事件驱动、非阻塞I/O模型。npm之生态系统是目前最大的开源包管理系统。

`——————命令操作—————

node –version 查看 node的版本号

npm –v 查看 npm 的版本号

cd 目录 : 切换目录

ls 查看当前目录下的文件和目录

mkdir + 文件名 :创建 目录

rmdir + 文件名 : 移除目录

cat 文件 :查看文件的内容

clear :清空当前控制台的命令`

什么是npm –>node package manager

npm init 初始化一个带有package.json的文件

管理本地安装的npm软件包的最佳方法 就是创建一个package.json文件
(npm init)

一个package.json文件:

1.它作为你的项目所依赖的包的文档
2.它允许你来使用指定版本的包
3.使你的构建可复用,这意味着它的方式更容易与其他开发人员共享

npm init –yes 将默认值直接创建,不会问你任何内容

npm install jquery –save 下载jquery依赖包 –save 保存到本地 本地依赖 整个项目必须依赖的模块

npm install swiper –save-dev 开发者依赖,

使用第三方框架的使用可以使用–save-dev

如果您的package.json目录中有文件并运行 npm install，那么npm将查看该文件中列出的依赖关系，并下载 所有这些文件的满足主要规则的最新版本
